# Task 3

Create a function that takes a number and return factorial of that number.

# Uruchomienie

Wejsc do pliku `factorial.hs` z Emacsa, i `C-c C-l`

Po pojawieniu sie REPL-a wpisac:

```
fact1 someNum
```

lub

```
fact2 someNum
```

i voila.
