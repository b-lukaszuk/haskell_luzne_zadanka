# Uruchomienie

Wejsc do pliku `*.hs` z Emacsa, i `C-c C-l`

Po pojawieniu sie REPL-a wpisac:

```
functionOfInterest [args]
```

Alternatywnie

```bash
runhaskell *.hs
```

i voila.

Najlepiej sprawdzic w pliku README.md danego zadania.
